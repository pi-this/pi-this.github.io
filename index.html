What is Pi-this?

My username on GitHub is Pi-this. I've used it to create a GitHub page site with the same name - pi-this.github.io. On this site, you can find all my repositories and projects. I share information related to my interests, which include computer science, math, STEM, and STEAM. Although I cannot include everything, I try to add more information to the website with each new version. The recent update, version 2.0, brings a fresh and colorful CSS style that enhances the site's professional look and feel. In this version, I’ve rolled out a more advanced user profile feature, which is unique in that all of your data is stored directly on your computer. This means you have complete control over your information, as it remains local rather than being reliant on external storage methods. Because the profiles are set up this way, they are private profiles. No one public can see this information, not even me. (pi-this) Additionally, the website is built and hosted through GitHub, specifically on GitHub Pages (github.io). This arrangement means that GitHub manages the hosting environment and is responsible for the security of the site. By utilizing their infrastructure, we can ensure a more reliable and secure experience for all users. Overall, this update focuses on improving user experience while prioritizing data privacy and security.

Why Pi-this?

Pi is an infinite decimal that represents the ratio of a circle's circumference to its diameter. I find pi fascinating, not only as a number but also as a symbol of endlessness. The website "pi-this" highlights the boundless nature of creativity, which is why I chose to use the word pi. If pi can represent endlessness, then creativity can also be described using the word pi. I even coined terms like "pi-this," "pi-that," and "pi-coins" just for fun. Anything on the site can have "pi-" added to the beginning. After all, pi is the limit!
