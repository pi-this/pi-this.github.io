Definitions

Pi

Pi is a number. Although it's length is long, it's value is only a little above three. The number's first fifteen didgets are: 3.14159265358979

get a more detailed definition on the word pi by looking at the merriam webster dictionary.

Raspberry Pi

The raspberry pi is not a pie with raspberries inside, that's a raspberry pie, with an E. The raspberry pi is a small pocket sized commuter that costs less than any other normal computer. This computer is there to help the newer generation as a learning tool.

To learn more about the raspberry pi go to raspberrypi.org.
